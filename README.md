
Апаратний MP3-плеєр на базі мікроконтролера STM32F401 з виводом високоякісного звуку через інтерфейс I2S та програмним декодуванням.

Про проєкт
Проєкт розроблено з метою адаптації класичних аудіосистем до сучасних цифрових форматів. Пристрій зчитує аудіофайли з SD-карти, програмно декодує MP3-потік за допомогою бібліотеки Helix та відправляє його на ЦАП PCM5102. Для забезпечення безперервного потокового відтворення без "заїкань" реалізовано Ping-Pong буферизацію через DMA у зв'язці з операційною системою реального часу FreeRTOS.

Архітектура
Система побудована на модульній основі з використанням багатозадачності:

Мікроконтролер: STM32F401CCU6 (Black Pill).

Аудіо-ЦАП: PCM5102 (підключено по I2S).

Зберігання даних: Модуль SD-карти (підключено по SPI).

Користувацький інтерфейс: OLED-екран (I2C) та апаратні кнопки.

Логіка роботи: Задача Audio_Task у FreeRTOS відповідає за читання файлів (FatFs) та декодування. Синхронізація передачі даних у ЦАП відбувається через апаратні переривання DMA (Circular Mode) та легкі сповіщення Task Notifications у FreeRTOS, що дозволяє мікроконтролеру ефективно розподіляти ресурси без блокування інших задач.

Технологічний стек
Апаратна платформа: STM32 (HAL, Bare-metal registers mapping для I2S PLL)

RTOS: FreeRTOS (Task Notifications, Memory Management)

Інтерфейси: I2S (з DMA Ping-Pong), SPI, I2C, UART (відлагодження)

Файлова система: FatFs

Декодер: Helix MP3 Decoder

Результати
Система забезпечує стабільне потокове відтворення аудіо. Вирішено проблему кліпінгу сигналу шляхом програмного даунскейлінгу амплітуди після декодування, а також налаштовано точну частоту дискретизації через ручне керування дільниками I2S.
