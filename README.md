Апаратний MP3-плеєр на базі мікроконтролера STM32F401 з виводом високоякісного звуку через інтерфейс I2S та програмним декодуванням.

Про проєкт
Проєкт розроблено з метою адаптації класичних аудіосистем до сучасних цифрових форматів. Пристрій зчитує аудіофайли з SD-карти, програмно декодує MP3-потік за допомогою бібліотеки Helix та відправляє його на ЦАП PCM5102. Для забезпечення безперервного потокового відтворення без "заїкань" реалізовано Ping-Pong буферизацію через DMA у зв'язці з операційною системою реального часу FreeRTOS.
